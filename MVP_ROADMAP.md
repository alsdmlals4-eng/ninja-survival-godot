# MVP_ROADMAP

## 현재 목표

Unity 원본을 참고해 Godot 4.x 기준으로 `닌자 서바이벌`의 최소 전투 루프를 재구현한다.

## MVP-001: Godot 최소 전투 루프

### 목표

Godot에서 플레이어 이동, 적 추적, 자동 투사체 공격, 점수 UI가 연결된 최소 플레이 루프를 만든다.

### 포함 범위

- 메인 테스트 씬
- 플레이어 8방향 이동
- 카메라 추적
- 하급 적 1종 스폰/추적
- 자동 수리검 또는 기본 투사체 공격
- 적 피격/제거
- 점수 증가
- 기본 게임오버 흐름

### 제외 범위

- Unity 코드 직접 변환
- 문파 선택
- 인법 조합
- 가방/휴식 페이즈
- 오의/보법 강화
- 상점/메타 성장
- 전체 애니메이션 세트
- 고급 무한맵

### 완료 기준

- Godot 프로젝트가 열린다.
- 메인 씬이 실행된다.
- 플레이어가 8방향으로 이동한다.
- 적이 플레이어를 향해 이동한다.
- 투사체가 자동으로 발사된다.
- 투사체가 적에게 맞으면 적이 제거되거나 피격 처리된다.
- 점수가 증가한다.
- 테스트하지 못한 항목은 보고된다.

## MVP-002: 성장 선택 최소화

MVP-001 완료 후 진행한다.

- 레벨업 또는 일정 점수 도달 시 성장 선택
- 공격 속도/투사체 수/이동 속도 중 일부만 적용

## MVP-003: 인법/장비 데이터화

- 무기/인법 JSON 데이터
- 기본 태그
- 간단한 시너지 표시

## MVP-004: 휴식/가방 프로토타입

- 전투 후 짧은 휴식 화면
- 장비 정리/선택 최소 기능
