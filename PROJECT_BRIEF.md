# PROJECT_BRIEF

## 프로젝트 한 줄 설명

`닌자 서바이벌`은 유파 인법을 모으고 백팩에 배치해 나만의 닌자 빌드를 완성하는 2D 서바이벌 로그라이크다.

## 장르

- 2D 탑다운 서바이벌
- Vampire Survivors 계열 자동 공격 로그라이트
- 인법 조합 빌드
- 백팩 배치 시너지
- 유파별 고유 규칙을 실험하는 반복형 로그라이크

## 플레이어 역할

플레이어는 네 닌자 유파 중 하나의 힘을 계승한 닌자다. 몰려오는 잠식 닌자와 요괴를 상대하면서 인법, 장비, 백팩 배치, 운명 선택으로 한 판의 빌드를 완성한다.

## 핵심 경험

전투 중에는 자동 인법과 처치 콤보로 즉시 쾌감을 얻고, 스테이지 종료 결과를 통해 무엇이 강했는지 확인한 뒤, 휴식 구간에서 백팩/조합/상점/운명 선택으로 다음 전투를 설계한다.

## 핵심 재미 우선순위

1. 인법 조합 빌드
2. 전투 쾌감
3. 백팩 배치 시너지
4. 유파 실험

## 핵심 루프

```text
유파 선택
→ 전투
→ 처치 콤보/보상 흡수
→ 중간보스
→ 스테이지 중간 결과
→ 휴식: 전리품/백팩/조합/상점/운명
→ 다음 전투
→ 최종 보스
→ 최종 결과/닌자소울
```

## 조작 방향

- 이동: 수동
- 기본 공격/전투 인법: 자동 발동
- 보법/오의: 수동 조작
- 휴식 구간: 가방 정리, 조합, 판매, 강화, 운명/미션 선택

## 4유파 MVP 방향

MVP에서도 4유파를 모두 넣는다. 단, 각 유파는 얕게 구현한다.

| 유파 | MVP 감정 | 최소 구성 |
|---|---|---|
| 봉마류 | 진지를 만들고 버티는 소환/결계형 닌자 | 공격형 식신, 영력 순환, 영력, 백귀야행, 소환+설치 시너지 |
| 천술류 | 속성 반응으로 화면을 제어하는 술법 닌자 | 화둔·염옥진, 오행순환, 오행 반응, 오행폭주, 젖음+감전 시너지 |
| 귀인류 | 위험하게 가까이 붙고 강해지는 근접 광전사 | 혈난무, 광전사, 귀혈, 귀인화, 근접+생존 시너지 |
| 흑영류 | 표식을 쌓고 터뜨리는 암살 닌자 | 만천화우, 암살교범, 암영표식, 암영처형, 원거리+암살 시너지 |

## Godot MVP 포커스

Unity 원본 전체 이식이 아니라, Godot에서 핵심 재미가 연결되는지를 단계적으로 검증한다.

포함:

- 기본 전투 기반
- 전투 DDD: 처치 콤보, 스타일리쉬 점수, 보상 흡수
- 4유파 얕은 구현
- 스테이지 중간 결과 화면
- 휴식 구간 흐름
- 백팩 기초와 조합 힌트
- 운명/미션 선택
- 최종 결과와 닌자소울 획득

제외:

- 4유파 전체 스킬풀
- 2차/3차 조합
- 복잡한 백팩 모양/회전/세트/저주 심화
- 완전한 상점 경제
- 정교한 엔딩 분기
- 화려한 UI/애니메이션 전체

## MVP 성공 질문

- 처치 콤보와 보상 흡수만으로 전투가 짧게 재미있는가?
- 4유파가 같은 스킨이 아니라 다른 플레이 감정으로 느껴지는가?
- 스테이지 종료 결과를 보고 무엇을 키울지 판단할 수 있는가?
- 백팩/조합/상점/운명 선택이 다음 전투 기대감으로 이어지는가?
- 최종 결과가 다음 판 동기를 주는가?
