# PROJECT_BRIEF

## 프로젝트 한 줄 설명

`닌자 서바이벌`은 닌자가 자동 공격과 수동 회피/오의를 활용해 요괴와 잠식 닌자의 공세를 버티고 성장하는 2D 탑다운 서바이벌 게임이다.

## 장르

- 2D 탑다운 서바이벌
- Vampire Survivors 계열 자동 공격 로그라이트
- 성장/빌드 선택 중심 액션

## 플레이어 역할

플레이어는 닌자 문파 중 하나에 속한 닌자로, 잠식된 닌자와 요괴의 공격 속에서 생존하고 인법을 성장시킨다.

## 핵심 경험

짧은 시간 안에 적을 처치하고, 자동 공격과 빌드 성장을 통해 점점 강해지는 도파민 루프를 제공한다.

## 핵심 루프

```text
전투 → 성장 선택 → 휴식/정리 → 더 강한 전투 → 보상/해금
```

## 조작 방향

- 이동: 수동
- 기본 공격/전투 인법: 자동 발동
- 보법/오의: 수동 조작
- 휴식 구간: 가방 정리, 조합, 판매, 강화, 미션 선택

## 첫 Godot MVP 포커스

Unity 원본 전체 이식이 아니라, Godot에서 최소 전투 루프를 먼저 검증한다.

포함:

- 플레이어 이동
- 카메라 추적
- 적 추적
- 자동 투사체 공격
- 피격/제거
- 점수 UI

제외:

- 문파 선택
- 인법 조합
- 가방 시스템
- 오의/보법 확장
- 휴식 페이즈
- 복잡한 애니메이션 전체
