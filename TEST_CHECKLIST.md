# TEST_CHECKLIST

## 공통 확인

- [ ] Godot 4.x에서 `project.godot`이 열린다.
- [ ] 실행 중 에디터 오류가 발생하지 않는다.
- [ ] 새로 만든 씬이 정상 실행된다.
- [ ] 테스트하지 못한 항목은 최종 보고에 명시한다.

## MVP-001 최소 전투 루프

### 플레이어

- [ ] 플레이어가 8방향으로 이동한다.
- [ ] 이동 속도가 과도하게 빠르거나 느리지 않다.
- [ ] 입력을 멈추면 플레이어가 정지한다.

### 카메라

- [ ] 카메라가 플레이어를 따라간다.
- [ ] 카메라 추적이 지나치게 흔들리지 않는다.

### 적

- [ ] 적이 플레이어 방향으로 이동한다.
- [ ] 적이 플레이어 주변에서 비정상적으로 떨리지 않는다.
- [ ] 적이 제거되거나 피격 처리된다.

### 자동 공격

- [ ] 투사체가 자동으로 발사된다.
- [ ] 투사체가 적 방향으로 이동한다.
- [ ] 투사체가 적에게 닿으면 충돌이 처리된다.
- [ ] 발사 간격이 지나치게 빠르거나 느리지 않다.

### 점수/UI

- [ ] 적 제거 시 점수가 증가한다.
- [ ] 점수 UI가 화면에 표시된다.
- [ ] UI가 게임 화면을 과도하게 가리지 않는다.

### 게임오버

- [ ] 플레이어 패배 조건이 있으면 게임오버가 표시된다.
- [ ] 재시작/메인 복귀 흐름이 있으면 동작한다.

## 저장/호환성

MVP-001에서는 저장 기능을 필수로 구현하지 않는다. 저장 기능을 추가하는 경우 다음을 확인한다.

- [ ] 저장 파일 생성 위치가 명확하다.
- [ ] 기존 저장 데이터가 깨지지 않는다.
- [ ] 저장 실패 시 오류가 보고된다.
